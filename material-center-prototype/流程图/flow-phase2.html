📊 素材报表（V1 基础底表）
成品素材投放后，数据回流至「素材报表 V1」。优化师在素材报表中查看基础底表数据，包含消耗、点击率、转化率、使用状态、关联广告数、视频完播指标等。V1 提供数据查看能力，交互功能（汇总/排序/导出）在 V2 开放。

🧩 三、模块定义（二期）
模块 核心功能 数据来源 产品要点
素材报表 V1（第一期） • 基础底表数据查看（表格视图）
• 日期筛选（上传日期/投放日期）
• 名称搜索筛选
• 全部数据列展示，包括：
  - 素材名称、素材类型、业务归属、渠道、优化师、设计师
  - 消耗、点击率、转化率、投放次数
  - 使用状态、关联广告数
  - 视频完播指标（仅视频素材显示）
• 数据列一次性全部展示（大数据已拉回全量）
不含：明细/汇总切换、列自定义、排序、维度切换、导出 		素材表 + 投放数据表 • 第二期只提供基础数据查看能力
• 所有数据列一次性展示（大数据已拉回全量）
• 交互功能在 V2 开放
• 不含汇总模式和导出功能
• 依赖投放数据 API 对接（消耗/点击/转化等数据回流）
